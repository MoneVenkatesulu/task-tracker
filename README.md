# Task Tracker

A simple full-stack **Task Management** application built with **React (Frontend)** and **Express + SQLite (Backend)**.  
It allows users to **add, view, update, and filter tasks**, and also provides **task insights**.

---

## Prerequisites

- Node.js >= 18.x
- npm >= 9.x

---

## Installation

### 1. Backend Setup

```bash
cd backend
npm install
Backend runs on: http://localhost:5000
```

### 2. Frontend Setup

```bash
cd frontend
npm install
Frontend runs on: http://localhost:5173
```
